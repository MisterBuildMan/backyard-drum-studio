# Backyard Drum Studio

A DIY backyard music studio build, documented from start to finish.

## Project Overview

This repository documents the complete process of building a sound-isolated music studio for acoustic drums in a residential backyard.

## Purpose

- **Primary Use:** Recording and practicing acoustic drums
- **Key Requirement:** Sound isolation (targeting STC 55-60)
