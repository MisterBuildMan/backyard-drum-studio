# Project Context

This is a DIY backyard drum studio build documentation project.

## Project Overview

- **Name:** Backyard Drum Studio
- **Purpose:** Sound-isolated space for acoustic drum recording and practice
- **Location:** Residential backyard
- **Foundation:** Concrete slab (to be poured)
- **Key Challenge:** Sound isolation (targeting STC 55-60)

## Repository Structure

- `specs/` - Phase specifications (requirements, design, tasks, materials)
- `designs/models/` - SketchUp files
- `images/` - Progress photos organized by phase
- `master-plan.md` - High-level phase tracking
- `CHANGELOG.md` - Progress log
- `glossary.md` - Construction terminology
