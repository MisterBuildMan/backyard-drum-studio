# Privacy Rules

This repository is public. Never add personally identifiable information (PII).

## Prohibited Content

- Street addresses (use `[address]` placeholder)
- Phone numbers
- Email addresses
- Full names of individuals
- Permit application numbers that could identify the property
- Photos showing house numbers, license plates, or neighbors' properties
- GPS coordinates or precise location data

## Allowed Content

- City/region name (e.g., "Cedar Park, Texas")
- Zoning district (public record)
- General lot size (e.g., "~0.3 acres")
- Building code references and section numbers
- Generic descriptions of property layout

## When in Doubt

Use placeholders like `[to be documented]` or `[address]` for any information that could identify the specific property or owner.
